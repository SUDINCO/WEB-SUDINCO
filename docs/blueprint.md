# **App Name**: CORE Login

## Core Features:

- Autenticación de Usuario: Autenticar usuarios basándose en su nombre de usuario y contraseña.
- Recuperación de Contraseña: Permitir a los usuarios recuperar su contraseña por correo electrónico.
- Formulario de Inicio de Sesión: Crear un formulario de inicio de sesión fácil de usar con campos de nombre de usuario y contraseña.

## Style Guidelines:

- Color primario: Azul para una sensación profesional y confiable.
- Color de fondo: Crema para crear una apariencia limpia y atractiva.
- Color de acento: Dorado oscuro para elementos interactivos, atrayendo la atención del usuario.
- Fuente del cuerpo y del encabezado: 'Inter', una sans-serif, para un texto limpio, moderno y legible.
- Utilizar iconos simples y consistentes para el nombre de usuario, la contraseña y otros elementos interactivos.
- Diseñar un diseño limpio y centrado para el formulario de inicio de sesión.
- Animaciones sutiles al pasar el ratón por encima de los botones y los campos del formulario para proporcionar retroalimentación al usuario.
